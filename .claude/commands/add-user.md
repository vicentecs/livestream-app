# Adicionar usuário de teste

Adiciona um novo e-mail à lista de usuários de teste no Google Cloud Console.

## Passos

1. Lembre ao usuário que isso é feito no Google Cloud Console, não no código
2. Forneça o link direto:
   `https://console.cloud.google.com/apis/credentials/consent`
3. Instruções:
   - Selecionar o projeto `livestream-orchestrator`
   - Seção **Usuários de teste**
   - Clicar em **Adicionar usuários**
   - Inserir o e-mail Google do novo usuário
   - Salvar

## Verificação

Após adicionar, o usuário deve conseguir logar em:
`https://live.seudominio.com` (ou IP local)

Nenhuma reinicialização do sistema é necessária.
